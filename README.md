# Dancing Chrome

음악의 음량, 저음, 박자에 맞춰 macOS의 Chrome 창을 움직이는 Manifest V3 확장 프로그램입니다. 작은 떨림부터 박자 순간이동, 화면 밖 이탈과 반대편 재등장까지 강도를 조절할 수 있습니다.

## 설치

### ZIP으로 설치

1. `Dancing-Chrome-v0.1.0.zip`의 압축을 풉니다.
2. Chrome 주소창에 `chrome://extensions`를 입력합니다.
3. 우측 상단의 **개발자 모드**를 켭니다.
4. **압축해제된 확장 프로그램을 로드합니다**를 누르고 압축을 푼 폴더를 선택합니다.
5. 툴바의 퍼즐 아이콘에서 **Dancing Chrome**을 고정합니다.

### 저장소에서 설치

이 저장소를 내려받은 뒤 위 2~5단계에서 저장소 루트 폴더를 선택하면 됩니다. 빌드나 패키지 설치는 필요하지 않습니다.

## 사용법

1. 음악을 재생하고 확장 프로그램 팝업을 엽니다.
2. **START**를 누릅니다.
3. Chrome 공유 창에서 **전체 화면**을 고르고 **오디오 공유**를 체크합니다. 이 방식은 여러 탭의 소리를 합쳐 감지합니다.
4. macOS가 화면 기록 권한을 요구하면 Chrome에 허용한 뒤 Chrome을 다시 실행합니다.
5. 다시 팝업을 열어 **STOP**을 누르면 멈춥니다.

전체 화면 오디오 공유가 제공되지 않는 macOS/Chrome 조합에서는 **현재 탭만 사용**을 누를 수 있습니다.

## 컨트롤

- **강도**: 1~87은 창을 화면 안에 유지합니다. 88 이상은 화면 밖 이탈 동작이 활성화됩니다.
- **감도**: 작은 소리와 박자를 얼마나 쉽게 움직임으로 인식할지 정합니다.
- **박자 점프**: 비트가 감지될 때 순간 이동하는 거리를 조절합니다.
- **RIGHT DRIFT**: 음악에 맞춰 오른쪽으로 이동하고, 완전히 사라진 뒤 화면 왼쪽에서 다시 등장합니다.
- **SAFE RETURN**: 중지할 때 창의 원래 위치와 크기를 복원합니다.

## 주의와 제한

- 최고 강도는 창이 화면 밖으로 나갑니다. 중요한 작업 중에는 낮은 강도로 먼저 시험하세요.
- Chrome 또는 macOS가 창 위치를 화면 안으로 강제 보정하는 환경에서는 화면 밖 이동 폭이 줄어들 수 있습니다.
- 현재 버전은 주 모니터 하나를 기준으로 계산하며 다중 모니터는 지원하지 않습니다.
- 최대화 또는 전체 화면 창은 시작할 때 일반 창 상태로 전환됩니다.
- 확장 프로그램은 오디오를 저장하거나 네트워크로 전송하지 않습니다. Web Audio API로 실시간 수치만 계산합니다.
