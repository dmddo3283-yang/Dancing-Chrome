# Dancing Chrome

재생 중인 음악의 음량·저음·박자에 맞춰 Chrome 창을 **화면 곳곳으로 날아다니게 하는** Manifest V3 확장 프로그램입니다. 관성을 타고 크게 휘어져 날다가 화면 가장자리에 부딪히면 튕겨 나가고, 비트가 칠 때마다 새로운 방향으로 솟구칩니다. 떨거나 진동하지 않고 시원하게 미끄러지듯 비행합니다. 소리가 멈추면 속도를 줄여 원래 자리로 돌아옵니다.

## 설치

### 저장소에서 설치 (권장)

1. 위 **ZIP 다운로드** 버튼으로 내려받아 압축을 풀거나, 저장소를 클론합니다.
2. Chrome 주소창에 `chrome://extensions`를 입력합니다.
3. 우측 상단의 **개발자 모드**를 켭니다.
4. **압축해제된 확장 프로그램을 로드**를 누르고 압축을 푼(또는 클론한) **폴더**를 선택합니다.
5. 툴바의 퍼즐 아이콘에서 **Dancing Chrome**을 고정합니다.

> 코드를 수정했다면 `chrome://extensions`에서 카드의 새로고침(⟳) 버튼을 눌러 다시 로드하세요. 별도 빌드나 `npm install`은 필요 없습니다.

## 사용법

1. 노래가 재생 중인 탭에서 확장 팝업을 엽니다.
2. 큰 **START** 버튼을 누릅니다. 지금 이 탭의 소리에 바로 연결됩니다 — `tabCapture` 기반이라 **선택 창도, 별도 권한도 필요 없이** 데스크톱 Chrome에서 곧바로 동작합니다.
3. 멈출 때는 팝업의 **STOP** 버튼, 또는 어디서든 단축키 **`Alt`+`Shift`+`X`** (창이 움직이는 도중에도 즉시 정지). 단축키는 `chrome://extensions/shortcuts`에서 바꿀 수 있습니다.

> Dancing Chrome은 "음악"을 따로 구분하지 않고, 그 탭에서 나는 **모든 소리**의 음량과 박자에 반응합니다.

## 컨트롤

- **강도** — 비행 속도·반경과 방향 전환의 날카로움을 한꺼번에 키웁니다. 낮으면 살랑살랑, 높으면 화면 전체를 가로질러 쌩쌩 납니다. (프리셋 표기: `HOVER · SWOOP · SOAR · WARP`)
  - 1–87: 창을 화면 안에 유지합니다.
  - 88 이상(**WARP**): 화면 가장자리 밖으로 벗어나는 비행이 허용됩니다.
- **감도** — 작은 소리와 박자를 얼마나 예민하게 움직임으로 인식할지 정합니다.
- **박자 반응** — 비트가 감지될 때 솟구치는 추진력(surge)의 크기를 조절합니다.
- **Flyby** — 켜면 좌우로 화면을 크게 휩쓸며, 가장자리에서 방향을 바꿔 계속 가로질러 날아다닙니다.
- **Size pulse** — 켜면 비트에 맞춰 창 크기(가로·세로)를 실제로 키웠다 줄였다 합니다.
- **Spin** — 켜면 그 탭의 페이지 화면을 CSS로 빙글빙글 회전시킵니다(실험적). 브라우저 창 자체는 회전할 수 없어, 탭 내부 콘텐츠를 회전시키는 방식입니다. 이를 위해 시작 시 해당 탭에 회전용 스크립트를 주입하며(`scripting` 권한), 중지하면 원래대로 되돌립니다. `chrome://` 등 일부 페이지에는 적용되지 않습니다.
- **Safe return** — 중지할 때 창의 원래 위치와 크기를 복원합니다.

## 동작 방식

- **오프스크린 문서**가 탭 오디오 스트림을 Web Audio API로 분석해 음량(RMS)·저음 대역·스펙트럼 플럭스 기반 박자를 계산합니다.
- **모션 엔진**은 속도·조향·항력·벽 반사로 이루어진 물리 비행 모델입니다. 목표점(웨이포인트)을 향해 가속하되 관성으로 지나쳐 휘어지고, 화면 가장자리에 부딪히면 튕겨 나가며 곳곳을 날아다닙니다. 소리가 멈추면 항력이 커져 속도를 줄이고 원위치로 착지합니다.

## 주의와 제한

- **WARP**(강도 88+)에서는 창이 화면 가장자리 밖으로 조금 나갑니다(항상 최소 절반 이상은 화면 안에 남습니다). 중요한 작업 중에는 낮은 강도로 먼저 시험하세요.
- `chrome.windows.update`는 창을 화면 밖으로 절반 넘게 옮길 수 없어, 창이 완전히 사라지는 연출은 불가능합니다.
- 현재 버전은 주 모니터 하나를 기준으로 계산하며 다중 모니터는 지원하지 않습니다.
- 최대화·전체 화면 창은 시작할 때 일반 창 상태로 전환됩니다.
- 오디오를 저장하거나 네트워크로 전송하지 않습니다. 실시간 수치만 계산합니다.

## 개발

```sh
npm test     # node --test (모션 엔진·박자 감지·설정·캡처 흐름 검증)
npm run check  # 주요 스크립트 문법 검사
```

외부 런타임 의존성이 없는 순수 HTML/CSS/JavaScript(Manifest V3) 프로젝트입니다.
